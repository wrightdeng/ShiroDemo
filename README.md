ShiroDemo
==============
一 项目简介
---------------
本项目主要是介绍Shiro权限管理框架的一些入门基础实例，以及如何将Shiro非入侵地整合到Spring、SpringMVC等WEB框架中。代码比较简单，同时有详细注释，供学习Shiro的新手朋友参考
